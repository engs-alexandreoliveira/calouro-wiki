# Programção e sistemas da computacionias I
- Programação I é a matéria que te ensina a pensar como programador, resolver problemas com lógica e transformar ideias em código usando estruturas básicas como variáveis, condições, laços e funções.

# Logica Operacional
- Logica operacional se trata de entender toda a logica de prohramação por trás de uma linguagem até o sistema construido, onde na matéria vemos contedudos como: Conjuntos, Subconjuntos, Logica Aristotelica, Logica proporsicional, entre putros conteudos que englobam a matematica discreta da computação

# Criatividade e Inovação
- Criatividade e inovação é a materia onde desenvolvemos senso criativo o suficentes para criação de ferramentas, produtos, emoreendimentos, que sejam relevantes para a sociedade, além de um senso critico de criação e disitnção de idaias aplicaveias ao mundo real

# Fundamentos para eng. software
- Os fundamentos para eng. software consistem em diversas praticas que tem finalidades em consolidar uma boa aplicação, onde na materia trabalhamos des de como configurar o git e criar uma conta no github, ate a conceeitos avançados de boas praticas dentro da engenharia de software

# Estatistica
- É a matéria que mostra como usar números e dados para entender melhor o que está acontecendo em um projeto de software, ajudando a analisar resultados e tomar decisões mais seguras.
